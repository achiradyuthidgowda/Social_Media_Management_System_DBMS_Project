# Social Media Management System — DBMS Mini Project

This project is a comprehensive Social Media Management System suitable for college submission, integrating DBMS concepts (ER, schema, normalization, SQL, constraints, triggers, stored procedures, etc.) with a real web frontend (HTML, CSS, JS, PHP). It manages Users, Posts, Comments, Likes, Followers, Messages, Notifications, and Admin features.

## Project Structure

```
/database
/sql
/documentation
/screenshots
/source_code
README.md
```

- **/database** — Diagrams, PDFs, images
- **/sql** — SQL scripts (schema, data, queries, etc.)
- **/documentation** — Report, abstract, viva, slides
- **/screenshots** — Add your real UI screenshots here after running locally
- **/source_code** — Full PHP/HTML/CSS/JS for app

## Quick Setup

1. Import `/sql/create_tables.sql` and `/sql/insert_values.sql` into MySQL
2. Edit `/source_code/config.php` for database settings
3. Run `/source_code` in local server (XAMPP/WAMP/LAMP)
4. Register/login and try all features, then capture real screenshots and insert them into `/screenshots` and documentation

## Features Covered
- Users, Posts, Comments, Likes, Followers/Following, Messages, Notifications, Admin
- Real-world table structure, constraints, PK/FK, joins, nested queries, views, indexes, triggers, procedures
- Clean commented code, modular UI, professional documentation
